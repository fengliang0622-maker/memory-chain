🌟 核心功能

1️⃣ 六层记忆架构

| 层级 | 名称 | 描述 |
|------|------|------|
| L1 | 初始记忆 | 碎片化信息，原始数据 |
| L2 | 情绪记忆 | 带情感色彩的经历 |
| L3 | 场景记忆 | 情境化存储，时空标记 |
| L4 | 语义记忆 | 抽象概念和知识体系 |
| L5 | 自我认知 | 身份认同，"我是谁" |
| L6 | 价值观与信念 | 人生观、世界观、核心价值观 |

2️⃣ 自动关联 (🔗)

发现并链接相关记忆
基于共同标签自动建立双向链接
构建知识图谱

3️⃣ 智能分类 (🏷️)

自动识别内容类型打标签：

重要 - 决策、关键信息

待办 - 需要完成的任务

经验 - 教训、总结

灵感 - 创意、想法

人物 - 主人、互动

项目 - 开发、功能

技术 - 代码、工具

生活 - 日常、习惯

4️⃣ 情绪检测 (😊)

自动识别情绪标签：

happy - 开心、高兴、喜欢、棒、帅
sad - 难过、伤心、失望
angry - 生气、烦、讨厌
fear - 害怕、担心
surprise - 意外、震惊

5️⃣ 语义检索 (🔍)

关键词搜索
层级过滤 (L1-L6)
标签过滤
情绪过滤
重要性过滤
时间范围过滤
访问频率排序

6️⃣ 摘要生成 (📝)

按时间周期 (7天/30天/自定义)
按层级分类统计
关键词提取
情绪统计
访问热度分析

7️⃣ 对话归档 (💬)

自动提取 #标签
识别决策/偏好/待办/经验
记录情绪倾向
场景上下文保存

8️⃣ 自动触发归档 (⏰)

触发条件：

会话结束关键词 (再见、拜拜、晚安)
1小时无对话
每天0点定时
