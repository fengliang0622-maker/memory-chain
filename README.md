Memory Chain - 六层记忆架构

基于人类记忆科学设计的AI人格化记忆系统，模拟从碎片信息到价值观形成的完整记忆过程。



功能特性



🧠 六层记忆架构 - 从初始记忆到价值观的完整体系

🔗 自动关联 - 发现并链接相关记忆，构建知识图谱

🏷️ 智能分类 - 自动识别内容类型打标签

🔍 语义检索 - 快速调取记忆，支持多维过滤

📝 摘要生成 - 结构化整理记忆，统计分析

💬 对话归档 - 自动提取对话关键信息



记忆层次




        

层级名称描述L1初始记忆碎片化信息，原始数据L2情绪记忆带情感色彩的经历L3场景记忆情境化存储，时空标记L4语义记忆抽象概念和知识体系L5自我认知身份认同，"我是谁"L6价值观与信念人生观、世界观、核心价值观
